# Coral Reef Health Classification

## Problem
Coral reef health is difficult to monitor at scale manually.

## Data
~1,000 labelled images across 3 classes: healthy, bleached, dead.

## Method
YOLOv8 transfer learning with image augmentation to address class imbalance.

## Results
80–90% classification accuracy across all 3 categories.

## Business Relevance
Enables low-cost automated reef health monitoring — scalable alternative 
to manual field surveys.
